## One-click install

### Claude (Web and Desktop)

[➕ Open in Claude and connect](https://claude.ai/customize/connectors?modal=add-custom-connector&mcpName=MFIT%20Personal&mcpServerUrl=https%3A%2F%2Fapi.mcp.ai%2Fp_mfit)

Manual: [claude.ai/customize/connectors](https://claude.ai/customize/connectors) → **+** → **Add custom connector** → name `MFIT Personal`, URL `https://api.mcp.ai/p_mfit`.

### Cursor

[➕ Install in Cursor](cursor://anysphere.cursor-deeplink/mcp/install?name=mfit&config=eyJ1cmwiOiJodHRwczovL2FwaS5tY3AuYWkvcF9tZml0In0=)
